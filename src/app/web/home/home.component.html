        ผมนายณัชพล สหัสสเนตร (ไมค์) ปัจจุบันศึกษาอยู่ที่ โรงเรียนทวีธาภิเศก open_in_new ชั้นมัธยมศึกษาปีที่ 6 สายการเรียน วิทยาศาสตร์ คณิตศาสตร์ คอมพิวเตอร์

        ตอนนี้ผมสามารถเขียน Web App ด้วย HTML CSS Javascript PHP ได้ หรือสามารถใช้ Framework เช่น Angular, Bootstrap เป็นต้น ส่วน Back-End สามารถเขียนด้วย NodeJs, PHP และ Database ใช้ MySQL, MongoDB ภาษาอื่นที่เขียนได้ Python และจะพยายามเรียนรู้ต่อไป เพราะ "การเรียนรู้ไม่มีที่สิ้นสุด"

        เว็บไซต์นี้พัฒนาด้วย Angular 13, Angular Material, NodeJS
